Simple.</p>\u000a\u000a<p>Here is Haskell:</p>\u000a\u000a<pre><code>quicksort :: Ord a =&gt; [a] -&gt; [a]\u000aquicksort []     = []\u000aquicksort (p:xs) = (quicksort lesser) ++ [p] ++ (quicksort greater)\u000a    where\u000a        lesser  = filter (&lt; p) xs\u000a        greater = filter (&gt;= p) xs\u000a\u000amain = do\u000a    file &lt;- readFile "data"\u000a    let un = lines file\u000a    let f = map (\u005cx -&gt; read x::Int ) un\u000a    let done = quicksort f\u000a    writeFile "sorted" (unlines (map show done))\u000a</code></pre>\u000a\u000a<p>And here is Python:</p>\u000a\u000a<pre class="lang-python prettyprint-override"><code>def qs(ar):\u000a    if len(ar) == 0:\u000a        return ar\u000a\u000a    p = ar[0]\u000a    return qs([i for i in ar if i &lt; p]) + [p] + qs([i for i in ar if i &gt; p])\u000a\u000a\u000adef read_file(fn):\u000a    f = open(fn)\u000a    data = f.read()\u000a    f.close()\u000a    return data\u000a\u000adef write_file(fn, data):\u000a    f = open('sorted', 'w')\u000a    f.write(data)\u000a    f.close()\u000a\u000a\u000adef main():\u000a    data = read_file('data')\u000a\u000a    lines = data.split('\u005cn')\u000a    lines = [int(l) for l in lines]\u000a\u000a    done = qs(lines)\u000a    done = [str(l) for l in done]\u000a\u000a    write_file('sorted', "\u005cn".join(done))\u000a\u000aif __name__ == '__main__':\u000a    main()\u000a</code></pre>\u000a\u000a<p>Very simple.
