If I do: </p>\u000a\u000a<pre><code>class A(object):\u000a    def __init__(self):\u000a        print "entering A"\u000a        print "leaving A"\u000a\u000aclass B(object):\u000a    def __init__(self):\u000a        print "entering B"\u000a        super(B, self).__init__()\u000a        print "leaving B"\u000a\u000aclass C(A, B):\u000a    def __init__(self):\u000a        print "entering c"\u000a        super(C, self).__init__()\u000a        print "leaving c"\u000a</code></pre>\u000a\u000a<p>Then I get: </p>\u000a\u000a<pre><code>entering c\u000aentering A\u000aleaving A\u000aleaving c\u000a</code></pre>\u000a\u000a<p>Note that <code>B</code>'s init never gets called.
